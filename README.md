# OpenGL Renderer

一个基于 Qt 和 OpenGL 的现代渲染器，正在开发中。

## 🚧 项目状态

**早期开发阶段** - 目前功能还不完备，基础框架不稳定。

## ✨ 当前功能

### 已实现的示例
1. **三角形** - 基础图元渲染
2. **矩形** - 四边形渲染
3. **三棱锥** - 简单3D物体渲染

### 核心特性
- ✅ 基于 Qt 的 OpenGL 上下文管理
- ✅ 现代 OpenGL (3.3 Core Profile)
- ✅ 着色器程序封装
- ✅ 基础图元渲染
- ✅ 线框/填充渲染模式切换
- 📋简单的相机系统

## 🛠 技术栈

- **C++17**
- **Qt 6**
- **OpenGL 3.3 Core Profile**
- **GLSL 330**

## 📦 构建说明

### 环境要求
- Qt 6.4.3 或更高版本
- C++17 编译器
- OpenGL 3.3 兼容显卡
